# Guía de Usuario - Secretaría

## Introducción

Esta guía describe las funciones principales del módulo de Secretaría en el Sistema de Gestión Escolar.

## Acceso al Sistema

1. Ingrese a `https://sistema.colegio.edu.gt`
2. Use sus credenciales institucionales
3. Será redirigido al panel de Secretaría

---

## Gestión de Pagos

### Crear Plan de Pagos

1. Vaya a **Pagos → Nuevo Plan**
2. Seleccione el estudiante inscrito
3. Ingrese el monto total y número de cuotas
4. Revise el desglose de cuotas
5. Haga clic en **Crear Plan**

### Registrar Pago

1. Vaya a **Pagos → Cuotas Pendientes**
2. Busque la cuota a pagar
3. Ingrese el monto y método de pago
4. Haga clic en **Registrar Pago**

### Ver Reporte de Morosos

1. Vaya a **Reportes → Morosos**
2. Filtre por grado (opcional)
3. Establezca monto mínimo (opcional)
4. Exporte a Excel si lo necesita

### Cierre Diario

1. Vaya a **Reportes → Cierre Diario**
2. Seleccione la fecha
3. Revise el resumen de pagos
4. Imprima para archivo

---

## Gestión de Contratos

### Generar Contrato

1. Vaya a **Contratos → Generar**
2. Seleccione la inscripción
3. Verifique los datos
4. Haga clic en **Generar**

### Ver Estado de Contratos

1. Vaya a **Contratos → Lista**
2. Filtre por estado (Pendiente/Firmado)
3. Haga clic en un contrato para ver detalles
